# CoOpen

## Описание
Днанная программа предназначена для одновременного открытия двух приложений.
## Особенности
Имеет интерфейс, сделанный с помощью SceneBuilder. Позволяет сохранять пути к исполняемым файлам. Сворачивается в трей.
## Установка и запуск
```
git clone https://github.com/BeSilent-q/CoOpen.git
```
Далее осуществить сборку через [launch4j](https://launch4j.sourceforge.net/).
### Важно!
Программа работает корректно только в собранном состоянии.
